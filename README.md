# Python-OpenCV Duygu Durum Analizi

![Sunum](https://i.ibb.co/sWXqWQb/splash.png)

## Proje Tanımı
Bir görüntü kaynağından elde edilen video ve fotoğraflar üzerindeki insan yüzlerinden duygu durum analizi yapan program.

